About DanceCat

DanceCat is a Data Reporting web tool by Python's Flask Framework. In version 0.1.0 you can:

  • Add, edit and test connections to databases.
  • Create and edit Query Data jobs which you can execute and store results to Redislite using Flask-Redislite. Results can be get later via DanceCat or sending direct email.
  • Scheduling Query Data jobs by Hourly, Daily, Weekly and Monthly. You can also schedule the job to be run once at some point.

Supporting Database:

  • MySQL using mysql-connector-python
  • SQL Server using pymssql

Develop environment:

  • Python: 2.7.9
  • Flask: 0.11.1
  • OS: *nix
