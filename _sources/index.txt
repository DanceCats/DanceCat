===============================
DanceCat: a Data Reporting tool
===============================

Introduction
------------

DanceCat is a simple data reporting tool which allowed user to get data and
have some basic overview on it.

Get the code
------------

DanceCat source code is always available on `GitHub <https://github.com/scattm/DanceCat>`_.
